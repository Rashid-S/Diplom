# Diplom
## Выполняемые функции
##### Данный код позволяет вывести список групп в VK в которых состоит пользователь, но не состоит никто из его друзей.
## Требования
##### Python 3.6 и выше
## Входные данные
##### 1. id пользователя
##### 2. Токен
## Выходные данные
##### Список групп заносится в файл result.json
